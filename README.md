# Smart Reminder Calendar

A beautiful and functional to-do list website with calendar integration and smart notifications.

## Features
- 📅 Interactive calendar view
- ⏰ Set custom date and time for tasks
- 🔔 Double notifications before expiration
- 🎯 Priority levels (High, Medium, Low)
- 📱 Responsive design
- 💾 Local storage persistence
- 🔍 Filter tasks by priority

## How to Use
1. Add a task with title, description, date, time, and priority
2. Set notification preferences (1-3 reminders)
3. Get notifications twice before task expiration
4. View tasks in calendar or list view
5. Click on calendar events for details
